# Builder.io

**Builder.io** is a US software company (visual CMS / frontend platform) that also maintains a strong open-source portfolio around high-performance web frameworks and, more recently, agent-native application infrastructure. It is the organization behind [[Agent-Native Applications|Agent-Native]].

## Notable open-source projects

- **[[Agent-Native Applications|Agent-Native]]** — framework for building apps where the AI agent and UI are equal citizens (this ingest).
- **Qwik** — resumable, fine-grained-lazy JS framework (ships near-zero JS initially).
- **Mitosis** — write components once, compile to React/Vue/Svelte/Angular/etc. (component transpiler).
- **Builder Visual CMS** — their commercial visual page builder.

## Relevance

Builder.io brings a frontend-framework pedigree to the agent space: Agent-Native's strong UI story (React + React Query + Tailwind + shadcn, optimistic UI, real-time collab) is unusual among agent frameworks, which tend to be backend/CLI-first. The "agent and UI as equal citizens" thesis is theirs.
