# NOTES — voicebox course

Scratchpad for user preferences and things to keep in mind when designing lessons.

## User profile (from mission interview, 2026-06-30)
- Beachhead: knows **React / TypeScript**. Start each new layer by connecting it to the web mental model.
- New layers: **Tauri/Rust**, **Python FastAPI backend**, **local ML model serving**. Don't assume any of these.
- Depth: deploy + principles, **no training/fine-tuning**.
- Env: **Mac (Apple Silicon)** → prefer MLX/MPS paths; treat CUDA/ROCm as conceptual only.
- Goal: build a similar app from scratch → bias every lesson toward "how would I build this," not "how does voicebox do it specifically."
