# NOTES — voicebox course

Scratchpad for user preferences and things to keep in mind when designing lessons.

## Teaching language
**Teaching language: 中文 (zh)** — 所有 lesson / reference / GLOSSARY 用中文写(2026-06-30 用户指定)。技术专有名词与代码保留原文(Tauri、FastAPI、`just dev`、sidecar 等),围绕它们的讲解用中文。

## User profile (from mission interview, 2026-06-30)
- Beachhead: 熟 **React / TypeScript**。每讲一层新东西,先从 web 心智模型搭过去。
- 新层:**Tauri/Rust**、**Python FastAPI 后端**、**本地 ML 模型部署**。这三层默认不会,不假设。
- 深度:部署 + 懂原理,**不训练/微调**。
- 环境:**Mac(Apple Silicon)** → 优先 MLX/MPS 路线;CUDA/ROCm 只讲概念。
- 目标:从零搭出类似应用 → 每节课偏向「我会怎么从零搭」,而不是「voicebox 具体怎么做的」。
