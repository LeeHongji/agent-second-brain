voicebox 的 sidecar 形状

你已经会 React 了 —— 很好,因为 voicebox 的界面本身就是一个 React 应用。让 voicebox 成为 voicebox 的,是这层 UI 周围的所有东西:一个桌面外壳(它捆绑了一个 Python 服务),而这个服务负责跑 AI 语音模型。本课要钉死整门课最重要的一个概念 —— sidecar 架构 —— 并把 voicebox 在你 Mac 上跑起来,这样后面每一课都有个真实的东西可指。

要背下来的那一张图

三个进程,一个应用。前端是你今天就会写的那部分;另外两个是你来这门课要学的。

┌──────────────────────────────────────────────────────────┐ │ TAURI 桌面应用 │ │ │ │ ┌──────────────┐ ┌───────────────────┐ │ │ │ React UI │ Tauri IPC │ Rust (src-tauri) │ │ │ │ app/ + web/ │◄────────────────►│ 窗口 · 托盘 · │ │ │ │ (这部分你 │ │ 热键 · 拉起 ───────┼─┐ │ │ 已经会) │ │ sidecar │ │ │ └──────────────┘ └───────────────────┘ │ │ └──────────────────────────────────────────────────────────┘ │ 启动并管理 ───┘ │ ▼ ┌──────────────────────────────────────────────────────────┐ │ PYTHON SIDECAR (FastAPI · uvicorn · :17493) │ │ backend/ routes · services · backends · database │ │ │ 加载并运行 │ │ torch · MLX ──► 7 个 TTS 引擎 · STT · 克隆 · LLM │ └──────────────────────────────────────────────────────────┘ (M9 时由 PyInstaller 打成单一二进制)

按"出声"的请求方向从右往左读这张图:UI 发请求 → Rust 外壳 → HTTP(:17493)→ Python → 跑模型 → 音频流回 UI。听写(输入)方向反过来。两种语言(Rust、Python)加上你会的 React,靠 localhost HTTP 这条边界粘在一起。

monorepo 目录地图

voicebox 是一个 Bun workspace。把每个目录对到上图:

  • app/ + web/ —— React + Vite 界面(你的舒适区)。
  • tauri/(src/ = Tauri 绑定的前端,src-tauri/ = Rust 外壳)—— 桌面那一层壳。
  • backend/ —— Python FastAPI sidecar(模型、路由、服务)。AI 就住在这里。
  • landing/ —— 营销站,我们不碰。
  • scripts/ + justfile —— 构建编排(sidecar 构建、dev、release)。

跑起来

仓库已经在 /Users/hongji/voicebox。项目用 justfile 作统一入口:

cd /Users/hongji/voicebox
just setup      # python venv + js 依赖(+ sidecar 前置)
just dev        # 确保 venv + sidecar,然后 `tauri dev`

如果你还没装 Tauri 前置依赖、just 或 bun,很正常 —— 装这些就是 M1 的一部分,也正是该来问我的地方。想先轻量看一眼(只看 UI、不碰外壳和模型):

bun run dev:web   # 只起 React UI,在浏览器里

本课的 tangible win

just dev 打开 voicebox 窗口。你看到一个你本来也能写出来的 React UI,被一个你还不会写(暂时)的 Rust 外壳托着。打几个字、挑个声音、让它说话。然后合上电脑,凭记忆画出上面那张 sidecar 图。两件事都能做到 = M1 完成:你既知道 voicebox 是什么,也知道后面每一课落在它的哪一层。

本课主源

扫一遍 voicebox 概览文档 和仓库里的 backend/README.md。现在不是研究功能,而是拿真实代码对照确认上面那个"三进程"形状。

这是对话,不是单向讲课。往下走之前随便问 —— 比如"为什么用 sidecar,而不是把 Python 直接嵌进 Rust?"、"IPC 和 HTTP 在这里各扮演什么角色、为什么两样都有?"、或"我跑 just dev 报错了"。当你能画出那张图、也听到它说话了,回一句 ready,我们就进 M2:Tauri 外壳本身。
